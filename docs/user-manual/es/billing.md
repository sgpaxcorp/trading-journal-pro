# Billing y planes
## Acceso
- Navegación lateral → Billing.

Billing es donde gestionas tu suscripción y add‑ons.

## Qué puedes hacer
1. Subir o bajar de plan.
2. Agregar o remover add‑ons (Option Flow, Broker Sync).
3. Cambiar entre mensual y anual.
4. Ver historial de pagos.

## Mejores prácticas
- Usa add‑ons solo si encajan con tu flujo.
- Alinea el ciclo de facturación con tu calendario de trading.
