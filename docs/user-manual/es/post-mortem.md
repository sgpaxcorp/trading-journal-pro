# Post-mortem (Option Flow)
Documenta lo que realmente paso despues del plan.

## Donde se usa
Ruta: `/option-flow`
Seccion: Post-mortem o Resultado real.

## Pasos
- Escribe que esperabas que pasara.
- Escribe que paso en realidad.
- Deja la leccion para la proxima sesion.

## Archivos adjuntos
- Puedes adjuntar imagenes para explicar el resultado.
- Tipos soportados: imagenes, HEIC, HEIF.

## Tips
- Se breve y honesto.
- Enfocate en el proceso, no solo en el resultado.

## FAQ
P: El post-mortem se guarda?
R: Aparece en el reporte despues de enviarlo.
