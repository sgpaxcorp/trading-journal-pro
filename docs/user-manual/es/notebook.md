# Notebook
## Acceso
- Navegación lateral → Notebook.

El Notebook es tu playbook. Guarda reglas, patrones, lecciones y mejoras que quieres repetir.

## Cómo se organiza
- **Notebooks**: contenedores principales (ej.: “Mi Playbook”, “Reglas de riesgo”).
- **Secciones**: categorías dentro del notebook (ej.: “Entradas”, “Salidas”).
- **Páginas**: notas individuales con contenido completo.

## Daily Notebook vs Custom Notebooks
- **Daily Notebook** para notas rápidas del día.
- **Custom Notebooks** para reglas estructuradas y playbooks de largo plazo.

## Mejores prácticas
1. Reglas cortas y accionables.
2. Actualiza después de la revisión semanal.
3. Documenta el “por qué” de cada regla.
