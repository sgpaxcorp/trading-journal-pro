# Reglas y alarmas
## Acceso
- Navegación lateral → Rules & Alarms.

Esta página convierte tus reglas en recordatorios y alertas automáticas.

## Recordatorios
Para hábitos diarios (premarket, journal, revisión).
1. Crea una regla de recordatorio.
2. Define el horario.
3. Actívala.

## Alarmas
Para señales críticas en vivo (riesgo, rompimientos).
1. Crea una regla de alarma.
2. Define la condición.
3. Actívala y verifica notificaciones.

## Mejores prácticas
- Mantén recordatorios simples y constantes.
- Usa alarmas solo para comportamientos críticos.
