# Biblioteca de recursos
## Acceso
- Navegación lateral → Resources → Library.

La biblioteca guarda todo tu material de aprendizaje en un solo lugar y lo conecta con tu proceso.

## Qué puedes guardar
- Videos de YouTube
- Libros y enlaces de Amazon
- Artículos
- Notas
- Links generales

## Cómo agregar un recurso
1. Elige el **Tipo**.
2. Escribe el **Título** (obligatorio).
3. Añade **URL** y **Autor** (opcional).
4. Escribe una **Nota** corta sobre por qué es relevante.

## Búsqueda y filtros
Usa búsqueda por título o contenido. Filtra por tipo para mantener orden.

## Exportar
Usa **Exportar CSV** para descargar tu biblioteca.

## Mejores prácticas
- Notas cortas y accionables.
- Conecta recursos con reglas en tu Notebook.
