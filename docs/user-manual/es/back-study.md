# Back‑Study
## Acceso
- Navegación lateral → Back‑Study.

Back‑Study es una revisión visual de tus trades. Sirve para validar entradas, salidas y calidad de ejecución.

## Cómo usarlo
1. Selecciona un rango de fechas o una sesión.
2. Revisa cada trade en el chart o timeline.
3. Compara la ejecución contra tus reglas.

## Tab de Auditoría
Dentro de Back‑Study usa Audit para revisar historial de órdenes (OCO, stops, cancelaciones). Es la capa de verdad de ejecución.

## Mejores prácticas
- Úsalo semanalmente.
- Enfócate en calidad de ejecución, no solo en resultados.
