# Ranking global
## Acceso
- Navegación lateral → Global Ranking.

El ranking global muestra los top traders según XP y trofeos. Se actualiza en vivo cuando se ganan trofeos.

## Qué verás
- **Top 25** con rango, nivel, XP y trofeos.
- **Tu snapshot** (XP, trofeos y si estás en el top 25).
- **Badges de nivel** (Bronce, Plata, Oro, Elite).

## Perfiles de trofeos
Haz clic en **“Ver trofeos”** para abrir el perfil público. Solo se muestran estadísticas públicas.

## Privacidad
Los perfiles muestran solo trofeos y stats públicos — no email, teléfono ni dirección.

## Mejores prácticas
- Usa el ranking como feedback, no como identidad.
- Los trofeos miden consistencia, no ego.
