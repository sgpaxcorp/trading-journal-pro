# Retos
## Acceso
- Navegación lateral → Challenges.

Los retos son programas cortos para construir consistencia y disciplina.

## Cómo usarlo
1. Elige un reto.
2. Confirma el inicio.
3. Da seguimiento desde el Dashboard y Journal.

## Mejores prácticas
- Toma un reto a la vez.
- Alinea el reto con tus metas del Growth Plan.
