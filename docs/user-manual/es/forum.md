# Comunidad (Foro)
## Acceso
- Navegación lateral → Forum → Community feed.

La comunidad es un espacio privado para usuarios logueados donde puedes hacer preguntas, compartir avances y revisar trades.

## Cómo navegar
1. Usa **Buscar** para filtrar por título o contenido.
2. Filtra por **Categoría** para mantener el foco.
3. Ordena por **Más activos**, **Más nuevos** o **Top (vistas)**.

## Crear un hilo (estándar de calidad)
1. Elige la categoría correcta (usa “Platform Questions” para temas de la app o imports).
2. El título debe tener mínimo 6 caracteres y ser específico.
3. El cuerpo debe tener mínimo 20 caracteres.
4. Explica contexto, qué pasó y qué esperabas.

## Respuestas
Abre un hilo y responde directamente. Las respuestas soportan Markdown (listas, links, tablas).

## Mejores prácticas
- Comparte evidencia cuando reportes un problema.
- Mantén feedback constructivo y específico.
- Usa el foro para mejorar proceso, no solo resultados.
