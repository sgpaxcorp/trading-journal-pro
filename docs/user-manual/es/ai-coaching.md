# AI Coaching
## Acceso
- Navegación lateral → Performance → AI Coaching.

AI Coaching convierte tu journal y analítica en feedback accionable. Funciona mejor con un número suficiente de sesiones.

## Qué lee AI Coaching
AI Coaching lee contexto conectado, no solo el chat.

Puede usar:
- notas del Journal
- trades y timing
- Neuro Layer
- Growth Plan
- analytics y KPIs
- patrones recientes de ejecución
- sesiones recuperadas desde todo el historial del journal dentro del rango seleccionado
- contexto automático de ejecución desde Audit cuando el lente o la pregunta lo requieren

Eso le permite comparar:
- lo que planificaste
- lo que hiciste
- lo que luego marcaste como verdadero
- cómo ese comportamiento impacta el próximo checkpoint del Growth Plan

## Cómo usarlo
1. Elige el lente de coaching.
2. Selecciona el rango de contexto que el coach debe revisar.
3. Usa la biblioteca de prompts si quieres arrancar más rápido. El dropdown configura la pregunta, el lente y el rango correcto en una sola selección.
4. Haz una pregunta específica (ej.: “¿Qué patrón me está costando más este mes?”).
5. Usa la respuesta para definir un cambio concreto para la próxima sesión.

Lentes de coaching disponibles:
- Rescate del plan
- Revisión semanal
- Disciplina de riesgo
- Verdad de ejecución
- Patrones psicológicos

Los prompts avanzados se dividen en dos grupos:
- prompts para rescatar el plan y proteger el próximo checkpoint
- prompts Neuro + ejecución para verdad operativa, drift emocional y foco pre-market

## Cómo cambia el coaching con Neuro Layer
Con Neuro Layer, AI Coaching puede detectar:
- entradas tempranas repetidas
- estados reactivos como urgencia o revancha
- salidas emocionales
- mismatch entre plan y verdad
- patrones personalizados que tú mismo creaste

Eso hace que el coaching sea más personal y más preciso.

## Mejores prácticas
- Ejecuta Audit primero si quieres el feedback más profundo basado en ejecución real. AI Coaching ahora puede adjuntar contexto automático de Audit, pero el Audit workbench completo sigue siendo la vista más rica.
- Usa `Rescate del plan` cuando la prioridad sea volver a alinearte con el Growth Plan.
- Usa `Verdad de ejecución` cuando quieras comparar el plan con los fills y el manejo real del trade.
- Pide una regla para agregar y una para remover.
- Revisa el coaching semanalmente para medir impacto.
- Mantén honestos los chips del Neuro Layer. El coach depende de buenos inputs.
- Revisa el dashboard después de una sesión de coaching. El último `Action Plan` aparece también allí como recordatorio operativo.
