# AI Coaching
## Acceso
- Navegación lateral → Performance → AI Coaching.

AI Coaching convierte tu journal y analítica en feedback accionable. Funciona mejor con un número suficiente de sesiones.

## Qué lee AI Coaching
AI Coaching lee contexto conectado, no solo el chat.

Puede usar:
- notas del Journal
- trades y timing
- Neuro Layer
- Growth Plan
- analytics y KPIs
- patrones recientes de ejecución

Eso le permite comparar:
- lo que planificaste
- lo que hiciste
- lo que luego marcaste como verdadero

## Cómo usarlo
1. Elige un rango de fechas con buena muestra.
2. Define el foco (disciplina, riesgo, ejecución o performance).
3. Haz preguntas específicas (ej.: “¿Qué patrón me está costando más este mes?”).
4. Guarda los action items y aplícalos en las próximas sesiones.

También puedes usar los atajos Neuro cuando quieras que el coach se enfoque específicamente en:
- drift repetido
- revisión Neuro reciente
- patrones de proceso de decisión

## Cómo cambia el coaching con Neuro Layer
Con Neuro Layer, AI Coaching puede detectar:
- entradas tempranas repetidas
- estados reactivos como urgencia o revancha
- salidas emocionales
- mismatch entre plan y verdad
- patrones personalizados que tú mismo creaste

Eso hace que el coaching sea más personal y más preciso.

## Mejores prácticas
- Ejecuta Audit primero si quieres feedback basado en ejecución real.
- Pide una regla para agregar y una para remover.
- Revisa el coaching semanalmente para medir impacto.
- Mantén honestos los chips del Neuro Layer. El coach depende de buenos inputs.
