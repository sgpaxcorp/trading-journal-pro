# Resources Library
## Access
- Left navigation → Resources → Library.

The Resources Library stores your learning material in one place so it stays connected to your trading process.

## What you can save
- YouTube videos
- Books and Amazon links
- Articles
- Notes
- General links

## How to add a resource
1. Choose a **Type**.
2. Enter a **Title** (required).
3. Add **URL** and **Author** (optional).
4. Write a short **Note** about why it matters.

## Search and filters
Use search to find by title or content. Filter by type to keep the library organized.

## Export
Use **Export CSV** to download your library for offline review.

## Best practices
- Keep notes short and actionable.
- Link resources to the rules you apply in your Notebook.
