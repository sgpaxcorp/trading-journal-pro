# Back‑Study
## Access
- Left navigation → Back‑Study.

Back‑Study is a visual replay of your trades. It helps you validate entries, exits, and execution quality.

## How to use it
1. Select a date range or session.
2. Review each trade on the chart or timeline.
3. Compare execution with your rules.

## Audit tab
Use the Audit tab inside Back‑Study to analyze order‑history behavior (OCO, stops, cancels). This gives objective execution truth.

## Best practices
- Use Back‑Study weekly.
- Focus on execution quality, not just outcomes.
