# Global Ranking
## Access
- Left navigation → Global Ranking.

The Global Ranking shows the top traders based on XP and trophies. It updates live as trophies are earned.

## What you see
- **Top 25 leaderboard** with rank, tier, XP, and trophy count.
- **Your snapshot** (XP, trophies, and whether you are in the top 25).
- **Tier badges** (Bronze, Silver, Gold, Elite).

## Trophy profiles
Click **“View trophies”** to open a public profile page. Only public stats and trophies are shown.

## Privacy
Profiles show trophies and ranking stats only — no email, phone, or address.

## Best practices
- Treat ranking as feedback, not identity.
- Use trophies to track consistency, not ego.
