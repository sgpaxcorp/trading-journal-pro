# Rules & Alarms
## Access
- Left navigation → Rules & Alarms.

This page enforces discipline with reminders and alerts. It is where your rules become automatic triggers.

## Reminders
Use reminders for routine behaviors (premarket, journaling, daily review).
1. Create a reminder rule.
2. Set the schedule.
3. Enable it.

## Alarms
Use alarms for live safety signals (risk limits, rule breaks).
1. Create an alarm rule.
2. Set the trigger condition.
3. Enable it and verify notifications.

## Best practices
- Keep reminders simple and consistent.
- Use alarms only for critical behaviors (max loss, rule violation).
