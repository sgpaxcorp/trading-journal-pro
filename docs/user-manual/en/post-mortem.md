# Post-mortem (Option Flow)

## Access
- Left navigation → Option Flow → Post‑mortem.

Document what actually happened after the plan.

## Where to use it
Use it inside Option Flow in the Post‑mortem (Real result) section.

## Steps
- Write what you expected to happen.
- Write what actually happened.
- Capture the lesson for the next session.

## Attachments
- You can attach images to explain the result.
- Supported types: image files, HEIC, HEIF.

## Tips
- Keep it short and honest.
- Focus on process, not only outcome.

## FAQ
Q: Is the post-mortem saved?
A: It appears in the report view after you submit it.
