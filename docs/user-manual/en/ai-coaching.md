# AI Coaching
## Access
- Left navigation → Performance → AI Coaching.

AI Coaching turns your journal and analytics into actionable feedback. It works best after you have a meaningful sample of sessions.

## What AI Coaching reads
AI Coaching reads connected context, not only your chat.

It can use:
- Journal notes
- trades and timing
- Neuro Layer
- Growth Plan
- analytics and KPIs
- recent execution patterns
- sessions retrieved from the full journal history inside the selected range
- automatic execution audit context when the coaching lens or question requires it

This is what allows the coach to compare:
- what you planned
- what you did
- what you later marked as true
- how that behavior affects your next Growth Plan checkpoint

## How to use it
1. Choose a coaching lens.
2. Select the context range the coach should search.
3. Use the prompt library if you want a faster start. The dropdown sets the question, lens, and range in one selection.
4. Ask a specific question (e.g., “What pattern is costing me the most this month?”).
5. Use the answer to define one concrete change for the next session.

Available coaching lenses:
- Plan rescue
- Weekly review
- Risk discipline
- Execution truth
- Psychology patterns

The advanced prompts are split into two groups:
- Growth Plan plays to protect the next checkpoint
- Neuro + execution plays for truth, emotional drift, and pre-market focus

## How Neuro Layer changes the coaching
With Neuro Layer, AI Coaching can detect:
- repeated early entry
- reactive states such as urgency or revenge
- emotional exits
- mismatch between plan and truth
- repeated custom patterns you created yourself

This makes the coaching more personal and more precise.

## Best practices
- Run Audit first if you want the deepest execution‑truth feedback. AI Coaching can now auto-attach available audit context, but the full Audit workbench still gives the richest view.
- Use `Plan rescue` when the priority is getting back on track with the Growth Plan.
- Use `Execution truth` when you want the coach to compare your plan against actual fills and trade management.
- Ask for one rule to add and one rule to remove.
- Revisit coaching weekly to track whether changes improved results.
- Keep Neuro Layer chips honest. The coach depends on good inputs.
- Check the dashboard after a coaching session. The latest `Action Plan` now appears there as an operating reminder.
