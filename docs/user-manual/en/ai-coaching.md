# AI Coaching
## Access
- Left navigation → Performance → AI Coaching.

AI Coaching turns your journal and analytics into actionable feedback. It works best after you have a meaningful sample of sessions.

## What AI Coaching reads
AI Coaching reads connected context, not only your chat.

It can use:
- Journal notes
- trades and timing
- Neuro Layer
- Growth Plan
- analytics and KPIs
- recent execution patterns

This is what allows the coach to compare:
- what you planned
- what you did
- what you later marked as true

## How to use it
1. Select a date range with enough sessions.
2. Choose a focus (discipline, risk, execution, or performance).
3. Ask specific questions (e.g., “What pattern is costing me the most this month?”).
4. Save the action items and apply them in your next sessions.

You can also use the Neuro shortcuts when you want the coach to focus specifically on:
- repeated drift
- recent Neuro review
- decision-process patterns

## How Neuro Layer changes the coaching
With Neuro Layer, AI Coaching can detect:
- repeated early entry
- reactive states such as urgency or revenge
- emotional exits
- mismatch between plan and truth
- repeated custom patterns you created yourself

This makes the coaching more personal and more precise.

## Best practices
- Run Audit first if you want execution‑truth feedback.
- Ask for one rule to add and one rule to remove.
- Revisit coaching weekly to track whether changes improved results.
- Keep Neuro Layer chips honest. The coach depends on good inputs.
